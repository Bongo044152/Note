## C語言包管理工具介紹

在其他語言中，我們常見的包管理工具有:

- python 的 pip
- node js 的 npm
- rust 的 cargo
- Java 的 Maven ( 好像是專案管理 )

... ( and more )

而對於 lunix 而言，由於有內建的系統套件管理員 `apt` 或 `yum` ... ( 根據發行商而有所變化，我目前只用過兩個 )，因此在引入C語言的函式庫會更佳容易。
